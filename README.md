# 🛡️ AI Powered Insider Threat Detection System

An AI-powered web application designed to monitor employee activities, analyze behavioral patterns, and detect potential insider threats within an organization. The system provides administrators with real-time insights through an interactive dashboard, helping improve organizational security.

---

## 📌 Overview

Insider threats are one of the biggest security challenges for modern organizations. This project uses Artificial Intelligence concepts and behavior analysis to identify suspicious employee activities and notify administrators before security incidents occur.

The application features a modern web interface with dashboards, reports, user monitoring, and threat visualization.
